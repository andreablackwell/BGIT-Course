Question 4: 
Describe the difference between inline elements and block-level elements in HTML. Give  examples of each.

Block elements in HTML begin on a new line and occupy the complete horizontal space of its parent element and have the same height as the content, for Example <div> About Me </div>. Inline elements in HTML do not begin from a new line and occupy space as required, for Example <a href="https://www.instagram.com/chocolate_b0mb" target=blank">Website</a>

Question 6: 
What are semantic HTML elements? Provide examples of semantic elements and explain  why they are important for web accessibility and SEO. 

Semantic HTML elements give meaning to the structure of web pages. They include elements like <header>, <nav>, <main>, <article>, <section>, <aside>, and <footer>. These elements help improve accessibility and SEO.

Question 7: 
What is the purpose of the HTML form element (<form>) and its associated form controls?  Provide examples of common form controls. 

HTML forms are essential for collecting user input on websites. They allow users to submit data, such as text, selections, and files. Forms are built using the <form> element, and various form elements like <input>, <textarea>, and <select> are used to create
different input fields.

Question 8: 
How can you add comments in HTML? Why are comments useful in web development? 

In HTML, a comment is a section of text that is not processed by the web browser. Comments are enclosed in <! -- --> tags. These tags tell the browser that the text inside them is a comment and should not be rendered on the front end.

Question 9: 
What is the HTML5 standard, and what are some of the new features introduced in HTML5?

HTML5 is the latest version of HTML. HTML5 introduced semantic elements, improved form elements, and native support for audio and video.
